# [[Markdown]] support
1. Code span 
`single quotes` 

2. Link Internally 
``` ## My Heading ## {#my-heading}```
then link normally as in [[Markdown]]

# Execution 
1. Create configuration file 
```doxygen -g <config-filename>```
Default filename is `Doxyfile`

2. Run doxygen 
```doxygen <config-filename>```
